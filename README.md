## Komutlar
Komut                   | Açıklama
----------------------- | ----------------------------------------    
`/start`                | Bota start verdiğinizde atılan ilk mesaj.
`/botsohbet`            | Size bir video gönderir.
`/selam`                | Fotoğraf gönderir.     
`/komut`                | Telegram adınızı gönderir.
